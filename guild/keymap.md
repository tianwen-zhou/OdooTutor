实用的 VS Code 快捷键
---------------

*   **F9**: 设置/取消断点
*   **F5**: 启动调试
*   **F10**: 单步执行（不进入函数）
*   **F11**: 单步进入（进入函数）
*   **Shift+F11**: 单步跳出（退出当前函数）
*   **Ctrl+Shift+F5**: 重启调试会话
*   **Shift+F5**: 停止调试


macOS: Shift + Option + ↓（向下复制） 或 Shift + Option + ↑（向上复制）
