### **1. 确保 VS Code 已正确安装**

首先，确认 **VS Code** 已经安装：

`open /Applications/Visual\ Studio\ Code.app` 

如果 VS Code 能正常打开，继续下一步。

----------

### **2. 在 VS Code 中启用 `code` 命令**

如果你之前没有设置 `code` 命令，可以手动启用它：

1.  **打开 VS Code**
    
2.  **按 `Cmd + Shift + P`** 打开命令面板
    
3.  输入 `Shell Command: Install 'code' command in PATH`
    
4.  **选择并执行**，然后重启终端
    
**然后重新尝试：**

`code .`
